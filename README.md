# 🚀 Personal Portfolio Website

A responsive personal portfolio website built to showcase my skills, projects, certifications, and journey as a Full Stack Developer. Designed with a modern dark/light theme, smooth animations, and a clean developer-focused layout.

🌐 **Live Demo:** [your-portfolio.vercel.app](https://your-portfolio.vercel.app)

## ✨ Features

- 🌙 Dark / Light mode toggle
- ⚡ Smooth animations with AOS and Framer Motion
- 📱 Fully responsive — mobile, tablet, and desktop
- 🧭 Smooth scrolling navigation with active section highlighting
- 🧑‍💻 Sections: Hero, About, Education, Experience, Skills, Projects, Certifications, Co-curricular, Contact
- 📄 Resume download option
- 🔗 Social media integration (GitHub, LinkedIn)
- 🏅 Certifications showcase with credential links
- 🚀 Projects section with live demo and GitHub links

---

## 🛠️ Tech Stack

- **React.js** (Vite)
- **JavaScript (ES6+)**
- **Tailwind CSS**
- **Framer Motion**
- **AOS** (Animate On Scroll)
- **React Icons**

## 🚀 Getting Started

### Prerequisites
- Node.js installed
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/rohit-pandey45/Personal-Portfolio.git

# Navigate to project directory
cd Personal-Portfolio

# Install dependencies
npm install

# Start development server
npm run dev
```

### Build for Production

```bash
npm run build
```
